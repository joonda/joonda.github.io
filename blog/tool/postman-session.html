Postman Cookie 자동 설정

8/6/2025

매번 설정하는 번거로움

포스트맨을 활용하여 API 테스트를 진행할 때, JSESSION Cookie를 활용하여 인증을 확인한다.
그러나, 재시작 등과 같은 이유로 세션 값이 바뀌는 경우가 자주 있는데 이 때마다 쿠키값을 설정해줘야하는 번거로움이 존재했다. 그래서 이를 트래킹해서 자동으로 설정해주는 방법이 없을까라는 생각을 하게 되었고, 설정하는 방법을 기록해놓고자 한다.

방법

포스트맨 쿠키 등록 장면

원래는 여기서 Add domain을 클릭하여 주소를 입력하고 (본인은 localhost 주소를 입력) 쿠키를 입력해주면 된다. 하지만 쿠키 값은 지속해서 바뀌기 때문에, 바뀔 때마다 입력해줘야하는 번거로움이 존재한다.

Postman Interceptor 해당 링크를 들어가보자.

Postman Interceptor Chrome web store

본인은 이미 확장 플러그인에 등록된 상태이다. 확장 플러그인으로 등록 한 뒤 플러그인에서 Sync Cookies를 클릭하자.

Postman Interceptor Plugin 그럼 해당 화면을 볼 수 있다. 여기서 본인이 원하는 도메인을 입력하자. 본인은 개발중인 프로젝트에서 포스트맨을 사용하기 위해 localhost로 등록했다.
