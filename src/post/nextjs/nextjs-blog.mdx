# NextJS 블로그 만들기

## NextJS로 블로그를 만들려는 이유
개발자라면 그래도 <b>블로그 하나쯤은 직접 만들어보고 싶다는 이유</b>에서였다. 아직 React도 제대로 많이 안다뤄본 나에게는 큰 도전이었다.  
이전에 티스토리, 네이버 블로그 등 하려는 시도는 많았는데 글을 쓴다는 것이 나에겐 너무 큰 부담이었던 것 같다.  

뭐 딱히 대단한 걸 쓰려는 건 아닌데 왜이렇게 부담을 많이 가질까 계속 생각해보면, 거창한걸 써야한다는 부담감 때문인 것 같다.
블로그를 내가 직접 만들면 애정을 가지게 될 것이고, 그러면 소소한 글이라고 쓰게되지 않을까 ? 라는 생각이 더 크게 작용해서 만들게 된 것 같다. (물론 Next.js도 써보고 싶은 이유도 있다)  

모르는 개념이 많아서 많이 헤맸고 어지저찌 만들었지만 당연하게도 보수할것이 아직 많이 보이는 것 같다, 그래도 배워나가다보면 많은 것이 바뀌지 않을까 라는 마음으로 정말 간단하게만 구현했다.  

그래서 디자인도 안했고, 글 목록, 글 상세페이지만 마크다운으로 간단하게만 구현한 것이다.  
어느정도 틀은 잡았고 시간도 많으니 부담갖지 말고 디자인도 해보고 많은 것을 넣어봐야겠다.  

레퍼런스는 [Tailwind blog](https://tailwind-nextjs-starter-blog.vercel.app/blog)로 삼았다, 내가 원하는 심플함이 딱 좋아서 참고를 많이 했다.
구현단계에서는 [김도형님 블로그](https://d5br5.dev/blog/nextjs_blog/setup)를 많이 참고했다, <del>이자리를 빌어서 너무 감사드립니다...</del>

어느정도 만들고 나니 많이 글을 써야지! 이 생각이 먼저 드는 것 같다. <del>(물론 언제까지 갈지는 모른다..)</del>
